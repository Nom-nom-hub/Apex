Modern. Fast. Scalable.

Apex is a full-stack web framework designed for building fast, scalable web applications with React and TypeScript.

Get Started

Features

File-based Routing

Automatic routing based on your file structure. No more manual route configuration.

Server-side Rendering

Blazing fast initial page loads with server-side rendering and client-side hydration.

Partial Hydration

Only hydrate interactive components, reducing JavaScript bundle size and improving performance.

TypeScript Support

First-class TypeScript support with excellent type inference and autocompletion.

Plugin System

Extensible plugin system for adding functionality like authentication, caching, and more.

Developer Experience

Built-in development server with hot reloading, error overlay, and more.

Getting Started

To create a new Apex project, run:

pnpm create apex-app@latest my-app

Then navigate to your project directory and start the development server:

cd my-app
pnpm dev
